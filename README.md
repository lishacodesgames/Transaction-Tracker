# Python Template
This project contains a simple main.py that prints "Hello World!" to act as a starting point for you project.

## Steps to use this template
After cloning, or clicking `Use Template` on GitHub,

1. List all your requirements in [requirements.txt](requirements.txt), each on a separate line
2. Run the *setup* script in `scripts/` that corresponds to your platform (`setup.bat` for Windows, `setup.sh` for Mac/Linux) <br>
**Make sure to run from a terminal!**
3. Activate venv in your terminal instance, the setup script will tell you the command :)

## Requirements
- Python 3+
- If on Linux, you will also require `python3.12-venv`
